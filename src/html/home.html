Welcome To Jct's Server

What is this?

This application is hosted in node.js, built by the head of the JCT Student Team. The core program is based around the module "Express", which can be found via npm. We chose Node.js because it offered a way to control how data comes into our system, and how our system deals with such data. Yes we could accomplish the same with Python or PHP, but we need to strive for modernism. All of the latest companies are pushing for node.js servers or apps, and with facebook pushing react native...
